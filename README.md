A responsive admin dashboard UI built as part of [The Odin Project](https://www.theodinproject.com/) Intermediate HTML & CSS course, focused on mastering CSS Grid for full-page layouts.


## Features

- **Full-page CSS Grid layout:** sidebar, header, and main content structured with nested grids.
- **Grid + Flexbox combo:** Grid for page structure, Flexbox for component-level alignment.
- **Custom project cards** with an inset box-shadow accent border.
- **Recolored SVG icons** via `filter: brightness(0) invert(1)` for the sidebar.

## Key Learnings

- **Grid vs. Flexbox:** Grid for 2D layouts (page skeleton, project cards grid), Flexbox for 1D alignment inside components.
- **Nested grids:** structuring a grid inside another grid taught me to think in layout layers, not one single grid.
- **Layout-first mindset:** planning the page structure before styling individual elements.
- **SVG icon handling:** finding, importing, and recoloring icons for consistent theming.

## How to Run Locally

```bash
git clone https://github.com/LucasAzevedoTheDev/admin-dashboard.git
cd admin-dashboard
```

Then just open `index.html` in your browser — no build step required.
